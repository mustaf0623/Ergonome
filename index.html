À propos d'Ergonome
⚡ Ergonome v7.4

Calculateur Woyofal intelligent pour les consommateurs sénégalais. Comprendre, suivre et optimiser sa consommation pour toujours acheter au meilleur tarif.

💱 Convertisseur (v7.4)

Deux sens : FCFA → kWh et kWh → FCFA, enrichis de la durée personnalisée basée sur ton profil calibré.

💰 Mes Économies — Nouveau v7.4

Panel proactif d'anticipation des économies. Génère des suggestions d'achat avant que le besoin se présente : montant optimal, meilleur moment, alerte solde critique, plan du mois, projection annuelle. Se déverrouille progressivement avec les données.

🎯 Planifier (v7.4)

Alimenté automatiquement par le Suivi et le profil calibré. Deux fonctions enrichies :

💰 Simuler : entre un montant → verdict + fourchette + durée personnalisée avec confiance.
📅 Plan de charge : entre ta conso/jour (pré-remplie) et les jours restants → montants recommandés + meilleur moment.

💾 Stockage IndexedDB

Les données sont dans IndexedDB — base de données intégrée, indépendante du cache navigateur. Fonctionne hors ligne.

🔒IndexedDB · Local · Hors ligne · Aucun serveur
